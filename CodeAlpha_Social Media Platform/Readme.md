# CodeAlpha – Social Media Platform

This is a **Mini Social Media Web Application**.

The project allows users to register, login, create posts, like posts, and interact through a simple social media interface similar to Instagram (basic version).

---

## Main File to Run

 **`frontend/auth.html`** is the **main entry point** of the application.

Open this file in a browser after starting the backend server.

---

## Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB (MongoDB Community / Compass)

##  How to Run the Project

### 1️.Clone the Repository


git clone https://github.com/vikas-6126/CodeAlpha-tasks.git


cd CodeAlpha-tasks/CodeAlpha_Social Media Platform

2. Run Backend Server
cd backend
npm install
npm start


You should see:



MongoDB Connected
Server running on port 5000

3. Run Frontend
Go to:
frontend/auth.html
✔ Double-click the file
✔ OR open it using Live Server in VS Code
